Rethomics

State of the art behavioural analysis can generate large amount of data. For instance, assume a simple experiment running 10 Ethoscopes for 1 week. In normal condition, we have two measurements a second (thins includes things like position and shape of the animal) on 20 animals. This results in a total of 10 * 20 * 2 * 3600 * 24 * 7 = which is actually 1/4 of a billion reads!

Whilst this is a large amount of data to deal with for graphical user interface based tool like Excel, this is a manageable job for statistical languages such as R. In order to facilitate data analysis, we have developed an R package named rethomics. It can be used for:

  • Loading data from the Ethoscope platform.
  • Visualising behavioural trends.
  • Analysing quiescence/sleep behaviour.
  • Analysing survival.

Rethomics has its own exhaustive documentation, and its own website.
