ethoscope.interactors.interactors module¶

class ethoscope.interactors.interactors.BaseInteractor(hardware_interface)[source]¶

Bases: ethoscope.utils.description.DescribedObject

Template class to interact with the tracked animal in a real-time feedback loop. Derived classes must have an attribute _hardwareInterfaceClass defining the class of the BaseInterface object (not on object) that instances will share with one another.

In addition, they must implement a _decide() method.

Parameters:hardware_interface (BaseInterface) – The hardware interface to use.
apply()[source]¶

Apply this interactor. This method will:

  1. check _tracker exists
  2. decide (_decide) whether to interact

3. if 2. pass the interaction arguments to the hardware interface :return:
